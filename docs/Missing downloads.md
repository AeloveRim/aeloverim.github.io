# Commonly Missing Downloads

The latest versions of wabbajack have trouble downloading mods from Loverslab. If you're getting messages about missing downloads after the installation ends, that's probably why. If so, you should download the following mods into your wabbajack download folder before starting the installer : https://rentry.co/x8e3vi5w
